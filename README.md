# VC Language Scanner

### Các thành viên trong nhóm:
- Lê Văn Thịnh 
- Lê Thanh Tùng
- Bùi Văn Phúc

### Prerequisites

- Java JDK8+

### Cách chạy chương trình

1. Xóa nội dung cũ trong file output/output.txt
2. Compile file Scanner.java
```javac Scanner.java```

1. Chạy file code vc<br>
```java Scanner <test_file_name>``` 
```ex: java Scanner input/input.vc```
